# Library Manager

Пакет Python для управления библиотекой книг с модульной структурой.

## Структура проекта
11_lesson_moduls/
├── library_manager/ ← основной пакет
│ ├── init.py
│ ├── catalog.py
│ ├── report.py
│ └── utils/ ← вспомогательные модули
│ ├── init.py
│ ├── data_validation.py
│ └── formatting.py
├── example_usage.py ← демонстрация работы пакета
└── README.md

## Демонстрация работы

**Файл `example_usage.py`** содержит полную демонстрацию всех возможностей пакета:

- ✅ Создание библиотеки
- ✅ Добавление книг с валидацией
- ✅ Генерация отчетов с датой и временем
- ✅ Поиск книг по различным критериям
- ✅ Удаление книг
- ✅ Статистика библиотеки

### Запуск демонстрации:
cd 11_lesson_moduls
python example_usage.py

## Описание модулей

### Основные модули

**`catalog.py`** - управление каталогом книг

- Класс `Library` - основной класс для работы с библиотекой
- Класс `Book` - представление книги
- Методы: добавление, удаление, поиск книг

**`report.py`** - генерация отчетов

- `generate_report()` - базовый отчет о книгах
- `generate_detailed_report()` - детализированный отчет с группировкой

### Вспомогательные модули (utils)

**`data_validation.py`** - валидация данных книг

- `validate_book_data()` - проверка корректности данных
- `validate_book_exists()` - проверка существования книги

**`formatting.py`** - форматирование данных

- `format_book_data()` - базовое форматирование
- `format_book_short()` - короткий формат
- `format_book_detailed()` - подробный формат

## Быстрый старт
from library_manager import Library, generate_report

### Создание библиотеки
library = Library()

# Добавление книг
library.add_book("Война и мир", "Лев Толстой", "Роман")
library.add_book("1984", "Джордж Оруэлл", "Антиутопия")

# Генерация отчета
print(generate_report(library))

# Поиск книг
books = library.search_by_author("Толстой")
